PART B: SHORT ANSWERS

1. Cardinality refers to the size or number of elements in a set. It can be used to determine whether two sets have the same number of elements (i.e., are equinumerous) or whether one set has more elements than another. Cardinality is denoted by |S| for a set S.

2. The Euclidean algorithm is a method for finding the greatest common divisor (GCD) of two integers. It works by repeatedly applying the formula gcd(a, b) = gcd(b, a mod b) until the remainder is 0. For example, to find the GCD of 24 and 36, we can apply the algorithm:
   gcd(24, 36) = gcd(36, 24) = gcd(24, 12) = gcd(12, 0) = 12

3. A reflexive relation R on a set S satisfies the property that for all x in S, (x, x) is in the relation R. An equivalence relation is a special type of relation that is reflexive, symmetric, and transitive. For example, the relation "is equal to" is an equivalence relation on the set of real numbers, while the relation "is less than or equal to" is a reflexive but not symmetric relation.

4. A direct proof starts with the given assumptions and uses logical reasoning to arrive at the conclusion. An indirect proof (or proof by contraposition) assumes the negation of the conclusion and then shows that this leads to a contradiction, thus proving the original statement. Direct proofs are typically used when the statement can be deduced step-by-step from the given information, while indirect proofs are useful when the direct approach is not straightforward.

5. A generating function is a way of encoding an infinite sequence of numbers into a single function. It can be used to solve recurrence relations by expressing the nth term of the sequence as the coefficient of x^n in the generating function. For example, the generating function for the Fibonacci sequence is F(x) = x / (1 - x - x^2).

6. In a depth-first search (DFS) of a binary tree, the algorithm visits a node, then recursively visits its left subtree, and finally its right subtree. The time complexity of DFS is O(n), where n is the number of nodes in the tree. In a breadth-first search (BFS), the algorithm visits all the nodes at the current level before moving on to the next level. The time complexity of BFS is also O(n).
