EXERCISE 1 (10 points)
Write a C function that takes an array of integers and its size as input, and returns the second-largest element in the array. If the array has less than two elements, return 0.

EXERCISE 2 (10 points)
Prove that the time complexity of the quicksort algorithm is O(n log n) in the average case, where n is the size of the input array.

EXERCISE 3 (10 points)
Compute the amount of memory required to store a structure that contains the following fields:
- an integer
- a float
- a character
- a double
- a pointer to an integer

Assume the system has a 64-bit architecture and uses standard C data alignment.
