** 1. FUNDAMENTAL CONCEPTS & DEFINITIONS **

*Core Terminology*
- Function: A relation between a set of inputs (the domain) and a set of permissible outputs (the codomain)
- Domain: The set of allowable input values for a function
- Codomain: The set of all possible output values for a function
- Range: The set of actual output values taken on by a function

*Conceptual Explanations*
- Functions model relationships between quantities
- Graphical representation shows how input and output are related
- Rates of change describe how a function's output varies with its input
- Limits describe the behavior of a function as its input approaches a particular value
- Derivatives measure the instantaneous rate of change of a function
- Integrals accumulate the net change of a function over an interval

*Key Principles & Theorems*
- Extreme Value Theorem: A continuous function on a closed interval attains its maximum and minimum values
- Intermediate Value Theorem: If a continuous function changes sign on an interval, it must take on the value 0 somewhere in that interval
- Mean Value Theorem: If a function is continuous on a closed interval [a,b] and differentiable on the open interval (a,b), then there exists at least one point c in the open interval (a,b) such that the derivative at c is equal to the average rate of change over the entire interval [a,b]

*Important Properties*
- One-to-one functions have unique inverses
- Even functions satisfy f(x) = f(-x)
- Odd functions satisfy f(x) = -f(-x)
- Periodic functions repeat their values at regular intervals

  - Domain: ℝ
  - Range: [-1, 1]
- Cosine: cos(x) = x/r, where r is the radius of the circle and x is the x-coordinate of the point on the unit circle

*Trigonometric Identities*
- sin²(x) + cos²(x) = 1
- tan(x) = sin(x)/cos(x)
- cot(x) = cos(x)/sin(x)
- sec(x) = 1/cos(x)
- csc(x) = 1/sin(x)
- sin(x + y) = sin(x)cos(y) + cos(x)sin(y)
- cos(x + y) = cos(x)cos(y) - sin(x)sin(y)
- tan(x + y) = (tan(x) + tan(y)) / (1 - tan(x)tan(y))

*Limits*
- lim_{x→a} f(x) = L if and only if for every ε > 0, there exists a δ > 0 such that if 0 < |x - a| < δ, then |f(x) - L| < ε.
- Properties of Limits:
  - lim_{x→a} [f(x) ± g(x)] = lim_{x→a} f(x) ± lim_{x→a} g(x)
  - lim_{x→a} [f(x)g(x)] = lim_{x→a} f(x) · lim_{x→a} g(x)
  - lim_{x→a} [f(x)/g(x)] = lim_{x→a} f(x) / lim_{x→a} g(x), provided lim_{x→a} g(x) ≠ 0

*Derivatives*
- Derivative of a constant: d/dx [c] = 0
- Derivative of a variable: d/dx [x] = 1
- Power Rule: d/dx [x^n] = nx^(n-1)
- Constant Multiple Rule: d/dx [cf(x)] = c·d/dx [f(x)]
- Sum Rule: d/dx [f(x) + g(x)] = d/dx [f(x)] + d/dx [g(x)]
- Difference Rule: d/dx [f(x) - g(x)] = d/dx [f(x)] - d/dx [g(x)]
- Product Rule: d/dx [f(x)g(x)] = f'(x)g(x) + f(x)g'(x)
- Quotient Rule: d/dx [f(x)/g(x)] = [g(x)f'(x) - f(x)g'(x)] / [g(x)]²
- Chain Rule: d/dx [f(g(x))] = f'(g(x))·g'(x)

*Integrals*
- ∫ c dx = cx + C, where C is an arbitrary constant
- ∫ x^n dx = (x^(n+1))/(n+1) + C, for n ≠ -1
- ∫ 1/x dx = ln|x| + C
- ∫ f(x) + g(x) dx = ∫ f(x) dx + ∫ g(x) dx
- ∫ cf(x) dx = c ∫ f(x) dx
- ∫ d/dx [f(x)] dx = f(x) + C
- ∫ sin(x) dx = -cos(x) + C
- ∫ cos(x) dx = sin(x) + C
- ∫ tan(x) dx = -ln(cos(x)) + C
- ∫ sec²(x) dx = tan(x) + C
- ∫ csc²(x) dx = -cot(x) + C
